## Proyecto destacado

### 🎟️ Ticketing Mundial 2026

Sistema de ticketing full-stack para el Mundial de Fútbol 2026: gestión de eventos, venta y transferencia de entradas, y validación de acceso mediante códigos QR.

- **Backend** — API REST en Spring Boot 3.3 (Java 21) sobre PostgreSQL, con autenticación JWT, triggers de integridad (límites de entradas y transferencias, control de superposición de eventos, validación de QR) y vistas de reporting (rankings de eventos y compradores, auditoría de funcionarios).
- **Frontend** — Next.js 16 (App Router) + React 19 + TypeScript, UI con Tailwind CSS y shadcn/ui, generación y escaneo de QR en tiempo real.
